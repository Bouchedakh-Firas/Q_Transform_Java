## Color Scheme
All buttons should use the following color scheme:

- Red Poppy: #f8485e (R 248, G 72, B 94)
- Dark Grey: #3c3c3a (R 60, G 60, B 58)
- White: #ffffff (R 255, G 255, B 255)

## Design Guidelines
- Button images should be clear and visually appealing
- Use the Red Poppy color as the primary color for buttons
- Use the Dark Grey color for text or secondary elements
- Use the White color for highlights or text on dark backgrounds
- Maintain consistent design across all button images
